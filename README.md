# Coding Agent Lab

这是一个用于学习、拆解、复现和比较各类 coding agent 的实验仓库。

当前重点：

- 记录不同 coding agent 的使用方式与设计思路
- 逐步复现 `OpenCode`、`Claude Code` 等工具的核心工作流
- 沉淀实验笔记、对比结论和可复用的实现片段

目录说明：

- `docs/`：概览、拆解记录、对比分析
- `reproductions/`：各个 agent 的复现代码与实验目录
- `notes/`：学习日志与阶段性结论
- `assets/`：截图、流程图等静态资源

后续计划会优先从最小可运行原型开始，再逐步补全工具调用、上下文管理、执行流程和安全边界。
