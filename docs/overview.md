# Overview

这里记录仓库的整体目标，以及各类学习文档的入口。

当前文档入口：
- [Claude Code 专题](./claude-code.md)：包含学习文档、最小实现 todo，以及对应 cleanroom 复现目录。
- [OpenCode 专题](./opencode.md)：记录 OpenCode 相关材料。
- [对比笔记](./comparisons.md)：整理不同 coding agent 在交互方式、工具边界和工程取舍上的差异。

当前仓库的文档重点是两类：
- 学习型文档：解释公开机制、实现边界和 cleanroom 推断范围。
- 进展型文档：同步复现代码当前做到哪里、下一阶段缺什么、验证方式是什么。
